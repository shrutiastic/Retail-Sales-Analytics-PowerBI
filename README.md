# Retail Sales Analytics Dashboard (Power BI)

## Project Overview

This project presents an interactive business intelligence dashboard built using Microsoft Power BI.
The dashboard analyzes retail sales data to identify trends in sales performance, product profitability, and regional market distribution.

The goal of this project is to help businesses make data-driven decisions regarding product strategy, regional performance, and profitability.

---

## Tools Used

* Microsoft Power BI
* Data Visualization
* DAX (Data Analysis Expressions)
* Business Intelligence Techniques

---

## Dashboard Pages

### Executive Overview

Displays key business metrics including total sales, profit, number of orders, and profit margin.
It also visualizes sales trends over time and category-wise sales distribution.

### Product Performance

Analyzes top-selling products, category profitability, and identifies products generating losses.

### Regional Analysis

Examines geographical sales distribution and highlights the most profitable regions and cities.

---

## Advanced Features Implemented

* Drill-Through Pages for detailed product analysis
* Dynamic KPI Indicators to monitor performance trends
* Interactive filters for category, region, and customer segment

---

## Dataset

Sample Superstore dataset containing retail transaction data including product details, sales, profit, region, and customer segments.

---

## Project Objective

To transform raw retail data into meaningful insights through interactive dashboards that support strategic business decisions.
